Variáveis

São espaços criados na mémoria

  • Podem
    • Podem começar com letras, $ ou _
    • Pode usar letras e números
    • Pode usar acenetos
  • Não Podem
    • Não podem começar com números
    • Não podem conter espaços
    • Não podem ser palavras reservadas
        
            //Declarando Variaveis
            var nome1 = "Gabriel"
            var nome2 = 'Gabriel'
            var nome3 = `Gabriel`

    Principais tipos de variáveis

  • Inteiro/Int: -50, -2, 0, 15, 200, 10000, 54000
  • Reais/Float/Double: -50.5, -9.34, 0.2, 15.34, 100.2345, 10000.324
  • Texto/String: "a", "Gabriel", "Nome Completo", 'Maria', `Artigo gigante de texto`, '123.654.987-96'. '(21)9 1234 - 9876'
  • Lógico/Boolean: true, false
